# 稀缺資源管理:時間與注意力(對應課程 Ch3)

關鍵心態:時間與注意力是**兩項不同的稀缺資源**,要分開管理。
你可能有時間卻沒注意力(累到無法專注),也可能注意力充沛卻把時間排給了低價值的事。

## 1. 兩項資源,分開盤點
- **時間**:總量固定,問題在「分配」。盤點一週時間實際花在哪(常和你以為的不同)。
- **注意力**:會被消耗、會被打斷。高品質注意力(深度專注)是稀缺中的稀缺,要刻意保護。

## 2. 制定每週計畫(工作 + 生活一起排)
- 不要只排工作待辦;把個人生活(運動、家庭、休息、學習)一起放進同一張優先順序表。
- 先放「重要但不緊急」的事(長期投資),否則它們永遠被緊急的事擠掉。
- 每週固定時間做計畫與回顧 —— 計畫本身就是一個迭代迴圈。

## 3. 情境轉換是注意力最大的敵人
- 每次在任務 / 情境間切換,都要付「重新進入狀態」的隱形成本;切換越頻繁,有效產能越低。
- 對策:
  - **批次處理**同類任務(集中回訊息 / 開會,而非散落整天)。
  - **保護深度工作時段**:畫出不被打斷的整塊時間給高價值、需專注的工作。
  - 減少並行:同時開太多任務 = 每個都做不好。

## 4. 最大化高效生產力
- 把高注意力時段對應到高價值 / 高難度工作;低能量時段做機械性雜務。
- 用「完成 vs 忙碌」自我檢查:今天推進了重要的事,還是只是反應式地忙?
- 生產力的槓桿在「做對的事」遠大於「做更多事」。

📤 產出:一週時間 / 注意力分配檢視 + 下週計畫(含深度工作時段與批次處理規則)。
