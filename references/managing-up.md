# 向上管理與升遷策略(對應課程 Ch4)

核心:升遷與影響力是**可設計的策略**,不是默默努力就會自動降臨,也不是逢迎拍馬。

## 1. 如何提出升遷:標準對焦 + 利害關係人遊說
- **標準對焦:** 先弄清楚升遷的「明文標準 + 潛規則」——這個層級實際被期待展現什麼?
  把自己的表現對齊到那套標準,而不是埋頭做自以為重要的事。
- **利害關係人遊說:** 升遷通常不是一個人說了算。盤點誰有影響力,提前讓關鍵人看見你的價值、
  在決策發生前就建立支持,而不是等評審當天才被認識。

## 2. 贏得關鍵人物支持
- 找出「決定你升遷的人」與「能影響他們的人」。
- 用對方在乎的語言溝通你的貢獻(對他們的目標 / 團隊的價值),而非只講你做了多少事。

## 3. 展現你對公司的價值
- 把工作成果**翻譯成商業 / 組織語言**:不是「我做了 A」,而是「A 帶來了 B(對公司目標的貢獻)」。
- 主動讓成果被看見(見 communication-influence.md),好工作不會自己發聲。

## 4. 克服冒牌者症候群(Imposter Syndrome)
- 先**正常化**:它極普遍,尤其在高成就者身上;有它不代表你不夠格。
- 用**證據**對抗自我懷疑:記錄具體成就與他人正回饋,在自我懷疑時拿出來對照。
- 別等「準備好了」才行動 —— 準備好的感覺常常不會先到;行動會帶來能力與信心。

## 5. 判斷是否該離開公司
理性框架,而非情緒衝動:
- 成長性:這裡還學得到東西、有上升空間嗎?
- 對齊度:工作內容與長期方向(見 career-design.md 的假設)是否一致?
- 環境:主管 / 文化 / 報酬是可改善還是結構性問題?
- 機會成本:留下 vs 離開,各自的下一版「你」會更好嗎?

## 6. 把自己打造成不被淘汰的產品 + 從失敗萃取成功因子
- **抗淘汰:** 持續迭代技能組合,讓自己的「產品價值」隨環境演進,而非依賴單一過時優勢。
- **失敗萃取:** 每次失敗做一次覆盤 —— 哪些是可控因子、哪些是運氣、下次哪個動作會不同。
  把失敗當資料,而不是判決。

📤 產出:升遷標準對照表 + 價值證據清單 + 關鍵人遊說計畫(或去留判斷的四面向評估)。
