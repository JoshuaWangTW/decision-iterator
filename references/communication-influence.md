# 溝通力與影響力(對應課程 Ch5)

核心:成果的影響力 = 成果品質 × 溝通效果。再好的工作,沒被對的人看見、聽懂,影響力趨近於零。

## 1. 冷啟動拓展人脈(Meta cold-start)
系統化、低壓力地建立關鍵連結,而不是靠尷尬的「social」:
- **以價值與好奇開場**,而非一上來就求助:請教、分享、給予,先建立關係再談需求。
- **小而持續**:定期接觸少數關鍵人,勝過一次性大量加好友。
- 把人脈當長期投資的網絡,不是有事才聯絡的工具。

## 2. 讓合作共贏(win-win)
- 先理解對方要什麼、在乎什麼,找出雙方都能得利的交集。
- 把「我要你幫我」重構成「我們一起達成各自的目標」。
- 長期關係的複利遠大於單次博弈的勝利。

## 3. 職場訊息撰寫(發什麼、怎麼發、發給誰)
- **發什麼:** 一則訊息一個明確目的 —— 是要決策、要資訊、還是知會?先講重點(結論先行)。
- **怎麼發:** 簡短、結構化、行動明確(對方讀完知道要做什麼、何時回)。
- **發給誰:** 收件人 vs 副本要分清(誰要行動、誰只是知會);別讓關鍵人淹沒在群發裡。

## 4. 高效會議管理(架構、角色、事後追蹤)
- **架構:** 每場會議有明確目的與議程;沒有目的的會議直接取消。
- **角色:** 誰主持、誰決策、誰記錄、誰只是與會——事先界定,避免「開完不知道誰要做什麼」。
- **事後追蹤:** 會後立刻發出 action items(誰、做什麼、何時),否則會議的價值會蒸發。
- 一句檢查:這場會議若改成一封訊息能不能解決?能的話就別開。

📤 產出:人脈經營計畫 / 一封結構化的職場訊息範本 / 一份會議議程 + action item 模板。
